# Spots

## Description
Spots is an interactive social media web application designed for sharing and viewing photos. The project features a responsive profile page that displays user information and a dynamic grid of photo cards. Each card contains an image, a title, and a like button. The layout is optimized to look great on all devices, from mobile phones to desktop computers, ensuring a seamless user experience across different screen sizes.

### Functionality
* **Responsive Layout:** The application adapts fluidly to various screen widths (mobile, tablet, and desktop) using CSS media queries.
* **Profile Section:** Displays user avatar, name, and description with "Edit Profile" and "New Post" buttons.
* **Image Grid:** A dynamic gallery of images that adjusts its column count based on the viewport width.
* **Interactive Elements:** Includes hover states for buttons and interactive "Like" buttons on photo cards.

## Technologies and Techniques Used
* **HTML5:** Semantic markup for accessibility and structure.
* **CSS3:** Advanced styling including Flexbox and CSS Grid for layout.
* **BEM Methodology:** Block Element Modifier naming convention for organized and maintainable CSS code.
* **Responsive Design:** Mobile-first approach using `@media` queries to handle breakpoints for varying screen sizes.
* **Fonts:** Custom "Poppins" font integration.
* **Normalize.css:** Used to ensure consistent rendering across different browsers.
